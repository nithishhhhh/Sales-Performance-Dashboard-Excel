 Sales Performance Dashboard (Excel)

This project presents an interactive Sales Performance Dashboard built using the Sample Superstore dataset.  
The dashboard analyzes sales, profit, customer segments, and product performance to generate business insights.

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- KPI Cards
- Data Visualization

---

## Dashboard KPIs
The dashboard tracks key business metrics:

- Total Sales
- Total Profit
- Profit Margin %
- Average Order Value
- Total Quantity Sold

---

## Analysis Performed

The dashboard includes the following analysis:

- Sales performance by Region
- Profit analysis by Customer Segment
- Category contribution to overall performance
- Top products by sales
- Interactive filtering using slicers (Region, Segment, Ship Mode)

---

## Key Insights

- West region generated the highest sales performance.
- Corporate segment showed the highest profit contribution.
- Office Supplies contributed the largest category share.
- Some segments show negative profit, indicating cost or discount impact.
- Overall profit margin remains low (~0.47%).
